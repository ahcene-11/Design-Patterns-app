# Vector Drawing Project

Mini-projet d’architecture logicielle (M1 GIL)

## Objectif

Création d’un éditeur de dessin vectoriel avec :

* édition via commandes
* sauvegarde XML
* fusion de dessins
* export PNG

## Architecture

Le projet utilise plusieurs design patterns :

* Composite (structure des formes)
* Visitor (opérations)
* Factory Method (création)
* Builder (chargement XML)
* Facade (API globale)
